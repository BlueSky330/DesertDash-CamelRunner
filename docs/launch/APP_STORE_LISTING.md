## Troubleshooting & Common Rejections

### "App contains hidden features not disclosed"
- Verify all features are listed in the description and What's New section
- Double-check that no ad networks or tracking methods are hidden

### "Misleading description or screenshots"
- Ensure screenshots accurately reflect gameplay
- Verify all claims (50+ skins, 10 countries, etc.) are implemented
- Remove any "Coming soon" features from the description

### "Inadequate privacy information"
- Include a link to the privacy policy in the description
- Verify the privacy policy URL is live and accessible
- Ensure privacy policy is up-to-date and mentions AdMob, Unity Ads, etc.

### "Requests unnecessary permissions"
- Double-check that the app only requests Internet permission
- Remove any unused permissions from Info.plist

### "Content inappropriate for age rating 4+"
- Verify no violence, profanity, or scary themes
- Ensure all ads are family-friendly (AdMob content rating: G)
- Check that in-game text and dialogue are age-appropriate
